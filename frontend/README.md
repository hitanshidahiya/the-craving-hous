# Craving Hous Frontend ☕

A modern and responsive frontend for the Craving Hous cafe ordering platform. This project focuses on delivering an engaging user experience for customers to explore the menu, customize orders, and place food orders seamlessly.

## 🚧 Project Status

This project is currently under active development.

The frontend is being built using React and Tailwind CSS. Some features, visual assets, and backend integration are still in progress and will be added in future updates.

## ✨ Current Features

* Modern and responsive user interface
* Cafe-inspired design and branding
* Interactive menu browsing experience
* Shopping cart functionality
* Multi-step order placement flow
* Customer information collection form
* Order summary and review section
* Mobile-friendly design
* Reusable React components

## 🛠️ Tech Stack

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* Vite

## 📦 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the frontend folder:

```bash
cd frontend
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
